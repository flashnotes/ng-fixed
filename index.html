ng-fixed


An AngularJS directive that changes an element's position to fixed when scrolled to a given point
This page demos a common case for this directive. It's a navigation menu that stayed pinned down and follows the user as they scroll down the page. Go ahead an give it a shot. You'll se the menu become fixed when you've scrolled down 200px.
